# 📚 E-Learning Platform

A simple and interactive E-Learning Platform built using **HTML**, **CSS**, and **JavaScript**. This project allows users to explore courses, enroll in courses, view lessons, and take quizzes in an easy-to-use interface.

## ✨ Features

- 📚 Course Listing System
- 🔍 Search Courses Functionality
- ✅ Course Enrollment System
- 📊 Interactive Dashboard
- 🎯 Quiz Functionality
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

- **HTML5** — Structure and content
- **CSS3** — Styling and responsiveness
- **JavaScript** — Interactivity and logic
